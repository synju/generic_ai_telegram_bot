1. Clone the repo
2. Run the virtual environment "venv_windows\Scripts\activate" or create your own "python -m venv venv_windows" and use it...
3. Install the requirements.txt "pip install -r requirements.txt"
4. Create a bot on telegram using the BotFather @BotFather and copy the Bot Token into the config
6. Get and OpenRouterAI API key by registering on openrouter.ai and copy the OpenRouterAI API key into the config.json
8. Under Bot Settings in the BotFather, TURN ON the bots "Allow Groups?" and TURN OFF the bots "Group Privacy"
9. Run the bot "python bot.py"
